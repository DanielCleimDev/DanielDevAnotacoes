Comandos Básicos Git

  • Para instalar o Git no Ubunto utilize o comando:

    sudo apt-get install git

  • Para instalar o GitHub cli com snap

    sudo snap install gh

  • Autenticando no github

    gh auth login

  • Comando para Criar um novo Repositório com flags para facilitar o uso:

    gh repo create nome-do-repositorio --public --description "descrição do repositorio"

  • Para inicializar um repositório Git local

    git init

  • Para Adicionar os arquivos modificados

    git add .

  • Para fazer um commit

    git commit -m "mensagem"

  • Adicionando repositório remoto

    git remote add origin git@github.com:DanielCleimDev/NOME-DO-REPOSITORIO.git

  • Configurando a branch como main

    git branch -M main

  • Enviando arquivos para o repositório remoto

    git push -u origin main

  • Clonando Repositório

    git clone git@github.com:DanielCleimDev/ -- NOME DO REPOSITÓRIO -- .git

  • Checa as alterações feitas

    git status

  • Baixar commits do repositório remoto para o repositório local

    git pull

  • Baixar arquivos da branch master

    git checkout master
